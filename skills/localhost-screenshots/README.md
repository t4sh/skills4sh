# localhost-screenshots

Take screenshots of locally running websites across all viewport breakpoints for visual comparison, regression testing, or documentation. Uses Chrome DevTools Protocol or Playwright depending on what's available.

## Install

```bash
npx skills add t4sh/skills4sh --skill localhost-screenshots
```

Installs to `~/.claude/skills/localhost-screenshots/`. Re-running is idempotent.

## Usage

See [`SKILL.md`](SKILL.md) for the full instruction surface the agent reads. Breakpoint presets, capture triggers, and diff workflows are documented there.
