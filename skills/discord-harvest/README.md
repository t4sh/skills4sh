# discord-harvest

Extract and download images, links, and files from Discord conversations — DMs via browser, channels via the Discord bot API. Handles stage-then-download flows, thread traversal, and content flagging.

## Install

```bash
npx skills add t4sh/skills4sh --skill discord-harvest
```

Installs to `~/.claude/skills/discord-harvest/`. Re-running is idempotent.

## Usage

See [`SKILL.md`](SKILL.md) for the full instruction surface the agent reads. Required sources, DM vs. channel workflows, and bot-token setup are documented there.
